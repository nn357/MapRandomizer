Room palettes

This setting controls the palettes for rooms (affecting foregrounds and backgrounds):

  • Vanilla: Rooms use the palette from the original game.

  • Area-themed: Each room uses a palette based on its area in the randomized map.

  • Area-shuffled: Each area is randomly assigned one of the six area palette themes, with every palette used exactly once.

  • Scrambled: Each room uses a random selection of one of six possible palettes.

  • If an individual area is selected (e.g. Crateria), then all rooms use a palette based on that area.
